# Hermes SMS Plugin for UISP

Send and receive SMS messages with your clients directly from the UISP admin panel, powered by Twilio.

## Features

- **Two-way SMS** — send messages to clients and receive their replies in a shared inbox
- **Conversation threads** — messages are grouped by contact in a chat-style UI
- **Client auto-match** — incoming messages are automatically matched to UISP client records by phone number
- **Client dropdown** — when composing a new message, select a client to auto-fill their phone number
- **Message pagination** — older messages are paginated so the page stays fast
- **Message log cap** — the message store automatically trims to the 2,000 most recent messages
- **Twilio signature validation** — webhook requests are verified to be genuinely from Twilio
- **Plugin logging** — errors and send/receive events are written to the UISP plugin log
- **Ctrl+Enter to send** — keyboard shortcut in the compose box

---

## Requirements

- UISP 2.1.0 or newer
- PHP 8.1+
- Composer
- A [Twilio](https://twilio.com) account with an SMS-capable phone number

```
README.md
src/
  manifest.json       Plugin metadata, configuration fields, and menu registration
  composer.json       PHP dependencies (UCRM Plugin SDK + Twilio SDK)
  composer.lock       Locked dependency versions (generated by composer install)
  main.php            Required UISP entry point (background tasks would go here)
  public.php          Main plugin page: SMS inbox UI + Twilio webhook handler
  .gitignore          Excludes vendor/ and data/messages.json from git
  data/
    .gitkeep          Keeps the data/ directory tracked by git
    messages.json     Created automatically at runtime — stores all messages
  vendor/             Created by `composer install` — not committed to git
```

## Setup

## Simple install 

Upload HermesPlugin.zip to the plugin area of UISP CRM and add your twilio info your done.

### 1. Install dependencies

From inside the `src/` directory:

```bash
composer install
```

### 2. Pack the plugin

From inside the `src/` directory:

```bash
./vendor/bin/pack-plugin
```

This creates `hermes.zip` one level up, next to this `README.md`.

### 3. Upload to UISP

1. Log in to UISP as an administrator
2. Go to **System → Plugins**
3. Click **Upload Plugin** and select `hermes.zip`
4. Enable the plugin

### 4. Configure Twilio credentials

1. In UISP go to **System → Plugins → Hermes → Configure**
2. Fill in:
   - **Twilio Account SID** — from your [Twilio Console](https://console.twilio.com) dashboard
   - **Twilio Auth Token** — from your Twilio Console dashboard (keep this secret)
   - **Twilio From Number** — your Twilio number in E.164 format, e.g. `+15551234567`
3. Save

### 5. Set the Twilio webhook

After the plugin is enabled and configured, the **SMS Inbox** page will display your webhook URL at the top. It will look like:

```
https://your-uisp.example.com/plugin/hermes/public?action=webhook
```

Copy this URL, then in the [Twilio Console](https://console.twilio.com):

1. Go to **Phone Numbers → Manage → Active Numbers**
2. Click your SMS-capable number
3. Under **Messaging Configuration → A message comes in**, set:
   - **Webhook** (HTTP POST) → paste your webhook URL
4. Save

Twilio will now POST to your UISP plugin whenever a client sends you a text.

## Opening SMS from a Client's Page

UCRM doesn't natively support plugin tabs inside the client detail page, but the plugin handles this with a smart redirect. When you're viewing a client in UISP, their ID is visible in the URL:

```
https://your-uisp.example.com/crm/client/123
```

You can jump directly to that client's SMS conversation by visiting:

```
https://your-uisp.example.com/plugin/hermes/public?clientId=123
```

- If that client has an existing conversation, you'll land directly on their thread.
- If not, the compose window opens pre-filled with their name and phone number.

**Tip:** Bookmark `?clientId=` as a browser bookmark with a keyword so you can type it quickly, or add it to a notes field on frequently messaged clients.



- Click **SMS Inbox** in the UISP sidebar
- Select a conversation from the left panel to view the thread
- Click **+ New** to start a conversation with any client or phone number
- Type your reply and press **Send** or **Ctrl+Enter**
- Incoming messages appear automatically (the page refreshes every 30 seconds)
- Click **View Client Profile →** in a conversation header to jump to that client's UISP page

## Troubleshooting

| Issue | Solution |
|---|---|
| "Access denied" on the plugin page | Make sure you are logged in to UISP as an admin |
| SMS not sending | Double-check your Twilio credentials in the plugin config |
| Incoming messages not appearing | Verify the webhook URL is set correctly in Twilio and that your UISP is publicly accessible over HTTPS |
| Twilio error 21211 | The destination number is not a valid E.164 phone number |
| Twilio error 21608 | Your Twilio trial account can only send to verified numbers |

Plugin errors are written to the UISP plugin log: **System → Plugins → Hermes → Log**.
